Getting Started

Prerequisites
    Node.js - v23.10.0
    npm - v10.9.2

Installation

Clone the repository:

git clone https://github.com/sangpetch-dev/music-playlist.git

cd music-playlist


Install dependencies:
npm install
# or
yarn install

Start the development server:
npm start
# or
yarn start

Open your browser and navigate to:
http://localhost:3000
